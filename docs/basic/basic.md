## 算法

一般而言，算法是解决问题的计算方法，是罗列出的解决步骤。

算法不等同于程序，我们可以说程序是算法用某种程序设计语言的具体实现。算法更关注于做什么，表现为一种思想。

部分内容来自于 [OI-Wiki](https://oi-wiki.org/)，那里也有许多算法的介绍，感兴趣的可以看看。
